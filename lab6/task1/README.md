Задание
---
Реализуйте множество с операциями «добавление ключа», «удаление ключа»,
«проверка существования ключа».

Input / Output
----

| Input      | Output     |
|------------|------------|
| 8          | Y |
| A 2 | N |
| A 5 | N |
| A 3 |  |
| ? 2 |  |
| ? 4 |  |
| A 2 |  |
| D 2 |  |
| ? 2 |  |
